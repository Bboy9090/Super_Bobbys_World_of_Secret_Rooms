### Step 1: Navigate to Folder
```powershell
cd C:\Users\Bobby\Documents\Bobbys-Secret-Rooms

```

### Step 2: Start Servers

**Option A: Double-Click Scripts**
- Double-click `START-BACKEND.ps1`
- Double-click `START-FRONTEND.ps1`

**Option B: PowerShell**
```powershell
.\START-BACKEND.ps1
.\START-FRONTEND.ps1
```

- Go to: **http://localhost:5000**
- Phoenix Key: `PHOENIX_RISES_2025`
