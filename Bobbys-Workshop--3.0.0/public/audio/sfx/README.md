## Audio Specifications

**Format**: MP3, 128kbps minimum
**Length**: 0.5-2 seconds per sound
**Volume**: Pre-normalized to prevent distortion
**Sample Rate**: 44.1kHz

## Creating Sound Effects

For a true 90s/00s Bronx workshop vibe, consider:

1. **Connection sounds**: Short beeps, subtle electronic tones
2. **Success chimes**: Positive 8-bit style or bell sounds
3. **Error sounds**: Buzzer or "whomp whomp" tones
4. **Clicks**: Mechanical switch or arcade button sounds
