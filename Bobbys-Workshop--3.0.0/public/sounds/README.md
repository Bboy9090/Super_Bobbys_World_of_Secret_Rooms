# Sound Assets Checklist (90s/00s Hip-Hop FX)

Place the following real MP3 files in this folder. Formats: MP3, 44.1kHz, ~128-192kbps, normalized to -3dB. No placeholders or silent files.

Required filenames:

- boom-bap-kick.mp3
- vinyl-scratch.mp3
- cassette-click.mp3
- air-horn.mp3
- cd-tray-open.mp3
- cd-tray-close.mp3
- sneaker-squeak.mp3
- basketball-bounce.mp3
- spray-can.mp3
- turntable-spin.mp3
- sample-chop.mp3
- record-drop.mp3

Usage notes:

- Sound manager auto-loads from /sounds/<name>.mp3
- Hooks: useSoundEffect('boom-bap-kick'), etc.
- Avoid long files (>2s) to keep UI snappy.
