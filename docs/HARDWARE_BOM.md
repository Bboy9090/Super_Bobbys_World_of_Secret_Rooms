## Hardware Compatibility Notes

### Platform Requirements

- **Windows**: All tools compatible with Windows 10/11
- **macOS**: Most tools compatible (some JTAG tools may require Windows)
- **Linux**: Full compatibility with most tools

### Driver Requirements

- **libusb**: Required for USB communication (included with BootForge)
- **FTDI Drivers**: Required for some JTAG tools
- **CH340/CH341 Drivers**: Required for some Chinese-made adapters
